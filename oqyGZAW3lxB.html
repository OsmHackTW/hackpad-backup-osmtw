標準問題/應對

  • 擺攤會遇到的問題
    • 有 API 可以用嗎?
    • OSM 是向量地圖嗎？
      •  2016 年最常被問到的問題
    • 這跟Google Map 有何不同?
    • 為什麼不用 Google Map?
      • (必答，請做好準備，很重要不必說三次)
      • 可參考 wiki 上的常見問題
  • 應對
    • 統一口徑，第一句先說明OSM是地理資料庫，而不是以地圖為中心來介紹下去( Yellowsoar 的 Idea )
    • 如果有可以先寫好話術會更容易同步

設備

  • 電腦 (Kagami)
  • 4G網卡 (Kagami)
  • 裕翔的滑鼠*1 (Kagami)
  • 大會攤位提供物品：
    • 報名成功錄取後，我們會提供更詳細的攤位規格書供大家參考。
    • 一張桌子（180 x 60 cm）且配有兩張椅子，有可能與其他社群共桌。
    • 至少一個 110V 電力插座（總消耗功率 550W 內），建議自行攜帶電力延長線。
    • 大會將會提供以 5GHz 為主之無線網路，但不保證順暢，請自行備妥離線方案。
  • 請確保大會期間之任何時段，攤位至少有一人顧攤。
