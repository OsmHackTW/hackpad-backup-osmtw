terminology

建議協會可以做一個對照表來定義一些專業名詞，如圖磚等等 https://www.facebook.com/groups/OpenStreetMap.TW/permalink/1084474604951341/

術語 簡介 詳細資訊
CartoCSS 圖磚樣式工具 CartoCSS+MML => Mapnik XML Config (NodeJS)
Cascadenik 圖磚樣式工具 Cascadenik+MML => Mapnik XML Config (Python) Cascadenik 屬性參考
changeset 變更組合，裡面會包含多筆點線面的異動，異動完成後需要做 close 動作表示完成
EPSG:4326 經緯度座標系
EPSG:900913 Google 投影座標系，900913 就是 Google 的意思，用於 Google Maps 和 Google Earth 服務 Google 地球投影
mapsforge Android/Java 平台的地圖輸出 API Android 地圖教學
mapnik 圖磚輸出工具
Maps.me 離線地圖 App iOS
mml Mapnik XML 造磚設定的半成品，需要用 Cascadenik 結合變數與 mss 樣式設定，拼裝成完整的造磚設定
mss Cascadenik 造磚樣式，通常用來管理顏色、線條、文字、以及圖資詳細程度
node
OruxMaps 離線地圖 App，而且可以編輯圖資，也支援 ANT+ 和 BLE 的運動感應器 Android
Protocolbuffer Binary Format (*.pbf) 二進位的圖資格式，佔用空間少，讀取速度快 (Protocolbuffer Binary Format) PBF_Format
POI 興趣點 (point of interest) 興趣點
PostGIS PostgreSQL 的外掛程式，增加了地理座標的資料型態，以及地理計算的函數
presets JOSM 翻譯為預設組合，意思是某一類圖資應該填寫的資料規範，例如 7-11 需要填寫 shop='convenience'
relation
Shapefile (*.shp) ESRI Shapefile 圖資格式，海岸線圖會用到
tile 圖磚，通常會做成 256x256 PNG 影像檔
Vespucci 圖資編輯程式，Android 平台 Android
WGS84 1984 年為 GPS 設計的座標系統 WGS84
WKT (well-known text) 的意思，一種純文字的幾何描述格式 Well-known text
Zoom 縮放等級 縮放等級，Zoom=N 表示把地表平面切割成 2^N x 2^N 等分，例如 Zoom=10 表示地表切割成 1024x1024 等分
Mapping Party 圖客趴、繪圖趴、繪圖活動、繪圖聚會
Mapper 圖客
tiles
