編修 Note 流程

以下流程使用iD編輯器！！！

  1. 開啟 OSM 網站：http://www.openstreetmap.org/#map=9/23.7414/120.5997；或直接開啟有 Note 圖層的 OSM 網站並跳到第三步驟：http://www.openstreetmap.org/#map=9/23.7414/120.5997&layers=N；（差別在於後者網址多了「&layers=N」）
  2. 開啟 Note 圖層：

  4. 點擊未解的 Note （紅色），查看內容。

  6. 確認 Note內容的資料正確性
    1. 搜尋引擎結果
    2. 門牌：全國門牌地址定位服務（TGOS）http://tgos.nat.gov.tw/tgos/Web/Address/TGOS_Address.aspx
    3. （此部分涉及科技應用能力，懇請支援補上各種可靠來源）
  7. 若確認 Note 內容無誤或已取得正確資訊，右鍵點擊左上角的「Edit」標籤，然後以新標籤頁打開連結編輯 OSM。注意：編輯時請盡可能只編輯 Note 內容，即本次變更組合（ChangeSet）只包含編修 Note 所需要的變更。請在提交變更組合的時候補充具體更改紀錄。

  9. 編輯完畢要上傳時，請複製 Note 的網址貼上到註解（comment）然後按下儲存。

  11. 上傳完畢後請右鍵點擊左側「View on OSM」圖示，並複製連結然後關閉分頁。

  13. 回到 Note 分頁，貼上剛剛複製的 Changeset 連結，然後按「Comment & Resolve」。

  15. 如果 Note 已經被編輯但沒有被解決掉，請比照第八步驟貼上相關的 Changeset 連結然後按下「Comment & Resolve」，如果 Changeset 內容太龐大請貼上相關的 OSM 物件連結。
  16. 如果 Note 資訊不足無法編輯，請在討論留下所缺乏的資訊，然後按下「Comment」。

  18. 以上如有不夠詳盡之處請再補述。
