土地測量、基石、三角點等標注方式

討論

Rex 個人偏向山頭基石分開標記，

1. 因為一個山頭有時好幾個基石。

2. 不同命名方便用戶遍識，如石門山目前共用基石、山頭。山頭仍為「石門山」、基石改為「石門山三等三角點」容易遍識。

範例

  • man_made=survey_point // 人工基石
  • checkpoint=hiking // 打卡點
  • name=* // 名稱
  • name:zh=* // 中文名稱
  • name:en= // 英文名稱
  • alt_name=* // 其他名稱
  • ele=* // 海拔
  • ref:survey_point= // 參考
