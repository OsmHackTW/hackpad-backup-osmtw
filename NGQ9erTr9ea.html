標注需求

  • 區分基石類型
  • 需要提供基石編號

標示範例

  • man_made=survey_point // 人工基石

  • checkpoint=hiking // 打卡點
  • checkpoint:type=survey_point
  • name=* // 名稱
  • name:zh=* // 中文名稱
  • name:en= // 英文名稱
  • alt_name=* // 其他名稱
  • ele=* // 海拔
  • ref:survey_point= // 參考
