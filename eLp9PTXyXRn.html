    • 不必要的 name tag. 譬如 "柏油路" "人行道" "路口"
    •  許多 way 用了過量的 nodes
    • tags 的使用也需要再檢視, 譬如 Texture=石頭 應該寫成 surface=*
    • ID, TerrainID, Build_Area, Build_NO, 等等 tag 可能要清掉
    • 有彎度的 way 也用了相當誇張數量的 nodes
    • 他無視前人已經畫的東西(譬如路, building), 直接重畫一個蓋上去, 但兩者有位移, 就看到兩個半疊在一起的東西
    • 他畫的很多東西直接重疊了兩次(甚至四次), 在原地 copy-paste 那種重疊, 所以不容易注意到。我清掉大部分了(試砍掉，常會發現砍掉之後原地還有一樣的東西)
    • 很多 highway=* 他都畫成 area, 不太有必要
    • 有些中文字的 encoding 壞掉了, 不是 utf-8
    • 可能是我 simplify nodes 時造成的(加上他 copy-paste 的因素), 相鄰的 landuse 原本共用座標的 nodes 被各自 simplify 成不同的座標, 導致相鄰的 landuse 相接得不太漂亮, 需要再修
    • 有些該接在一起的路沒接起來
    • 有橋的地方 waterway 不應該斷掉，應以 layer 處理
