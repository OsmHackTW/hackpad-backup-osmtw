標示方法

目前標示會分成兩大類別, 一類是直接標示飲水機/點, 另一類則是標示該建築物/區域有提供飲用水/機.

類別一：直接標示飲水機/點

  • name（「地標」+「飲水型態」，若無名字則填入「飲水點」或是把 tag 撈回來組成 name）
    • 飲水型態：飲水機、直飲台、生飲台（沒有濾心）
    • 加水站建議用 amenity=water_point（對象不是人）
    • 舉例：萬隆捷運站生飲台、警察局飲水機
  • amenity=drinking_water 用途、做什麼用，單獨標示飲水機用的，捷運站內的飲水機
    • http://wiki.openstreetmap.org/wiki/Tag:amenity%3Ddrinking_water
  • description 描述
    • 使用者自填 ex山區「最後水源」
    • 是否需要自備容器？重點在想描述出, 可"直接用嘴喝", 不需任何杯瓶 
      • 需自備容器/不需自備容器
      • 備容/免容
      • 免容器
      • 直接用嘴喝
  • source 資料來源
  • indoor=yes http://wiki.openstreetmap.org/wiki/Key:indoor
    • level=樓層（數字）http://wiki.openstreetmap.org/wiki/Key:level
    • access=* （例如環資屬於 permissive）
    • opening_hours 開放時間 http://wiki.openstreetmap.org/wiki/Key:opening_hours
  • brand（ex賀眾, 3M）
  • hot_water=yes
  • warm_water=yes
  • cold_water=yes（意思是冷水！並不是冰水）
  • bottle=yes（意思是可以很容易用瓶水裝水）
    • bottle=require（可以考慮新增）
    • https://taginfo.openstreetmap.org/keys/bottle#values

類別二：標示該建築物/區域有提供飲用水/機

  • name
    • 舉例：台灣環境資訊協會
  • drinking_water=yes/或其它（用在建築物內的飲水機）
  • 飲水提供方式比較不一定、裝水者需要詢問場地機關、可能地點本身有原先用途而飲水只是附屬
    • http://wiki.openstreetmap.org/wiki/Key:drinking_water
  • source 資料來源
  • description 描述 (這個類別，有辦法撰寫「描述」嗎？)
  • drinking_water:hot_water=yes
  • drinking_water:warm_water=yes
  • drinking_water:cold_water=yes
  • drinking_water:bottle=yes
  • indoor=yes http://wiki.openstreetmap.org/wiki/Key:indoor
    • level=樓層（數字）
    • access=* （例如環資屬於 permissive）
    • opening_hours 開放時間 http://wiki.openstreetmap.org/wiki/Key:opening_hours

  • 欄位要在對的地方
  • 地址轉換為經緯度
