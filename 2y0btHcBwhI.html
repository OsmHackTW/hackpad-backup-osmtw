自己的圖磚自己刻 docker 大法

建立資料庫位置

  • mkdir /tmp/osm-postgresql && chmod 777 /tmp/osm-postgresql
  • wget -O /tmp/import.osm http://download.geofabrik.de/asia/taiwan-latest.osm.pbf

建立伺服器

  • docker pull chihchun/openstreetmap-tiles
  • docker run -v /tmp/osm-postgresql:/var/lib/postgresql chihchun/openstreetmap-tiles initdb startdb createuser createdb 
  • docker run -v /tmp/osm-postgresql:/var/lib/postgresql -v /tmp:/data chihchun/openstreetmap-tiles startdb import
  • docker run -P -v /tmp/osm-postgresql:/var/lib/postgresql chihchun/openstreetmap-tiles startdb startservices
  • docker run -p 127.0.0.1:8080:80 -P -v /tmp/osm-postgresql:/var/lib/postgresql chihchun/openstreetmap-tiles startdb startservices

firefox http://localhost/8080

除錯

  • docker run -v /tmp/osm-postgresql:/var/lib/postgresql -v /tmp:/data -i -t homme/openstreetmap-tiles cli
