傳統信仰標注方式 (草案)

三、四月份在 OSM 台北聚會 (3/21) 的時候，社群曾經討論過傳統信仰標注的問題[2]。在台灣地區的地圖上，已經被標注了很多廟宇，但是志工在標注的時候難以以現有的 (道教、佛教) 地圖標籤系統[3][4] 歸類。社群希望能夠討論出一個新的標注系統，來改善現有使用方式。

例如以財團法人台北市艋舺龍山寺[5]為例子，在內政部網站[6]上是被歸類為佛教。但目前龍山寺並非佛寺，祭拜的神佛與儀式也無法直接歸內為道教。目前在 OSM 上面是同時標注為 buddhist;taoist;folk 三種信仰系統[7]。 

  

由於台灣傳統宗教類型為儒、道、釋、傳統信仰等幾個信仰混合，但是目前主流標籤使用為 religion=taoist 或 religion=buddhist ，無法精確的呈現台灣傳統信仰的多元性[3]。

標注方式

比較精確的方法或許是而道觀、佛寺等建築或場所保持原本的  religion=taoist , religion=buddhist 標籤。

民間信仰標注

民間信仰是一個普遍的信仰系統，在所有國家都有其民間信仰系統[1][2][3][4][5]。因此直接使用 religion=folk 即可。 

但是於台灣地理區域內，不只只有台灣漢人的民間信仰，例如凱達格蘭族的七星山祭壇[6]或西拉雅族[8]的公廨也屬於民間信仰。

  

台灣漢人傳統信仰標注為

religion=folk

denomination=TaiwaneseFolkReligion

 而西拉雅族祭拜阿立祖的公廨可標注為

religion=folk

denomination=SirayaFolkReligion 

這個標注方法會造成前端的一些困擾，例如只有在 denomination=TaiwaneseFolkReligion 在對應出台灣傳統廟宇的圖徵 (求圖示設計)。

  • 歸類的判斷方式
    •
    • 如寺廟或道壇有神職人員道士或法師，則歸類道教。標注為 religion=taoist.
    • 如只有乩童或是只廟公、管理委員會，但無神職人員的話，就歸類為台灣傳統信仰。
    • 大樹公如果設有固定香爐 (如水泥)，可標注為台灣傳統信仰。
    • 厭勝物 可標注為台灣傳統信仰

    建築物標記方式

    另外將傳統信仰的寺廟、道壇、鸞堂分成新的類別

    • 寺：原指官舍，如太常寺等，佛教傳入中國後中國方將僧人居所改稱為寺[1]。
    • 廟：古時多指奉祀祖先與先聖先賢的宗廟，有太廟、文廟、武廟、家廟等，後來廣泛應用於祭祀其他神祇的宗教建築，如城隍廟、媽祖廟等等[1]。
    • 壇：為祭天地之處，傳統的壇為高臺，有天壇、地壇、社稷壇、山川壇與風雲雨壇等等[1]。
    • 祠：多指祭祀祖宗或先賢等等的建築，如宗祠、名宦祠等[1]。
    • 觀：原指宮殿的高大門闕，後來用以指道士修道的居所，即道觀[1]。
    • 殿：指帝王居所或供佛寺神的屋宇[1]。
    • 堂：原指一組建築居中之處的正堂，後常用於齋教的宗教建築，即齋堂[1]。
    • 院：原為官廨別稱，後為寺廟與書院使用[1]。
    • 庵：本來是指結草為屋，後用來指僧尼供佛的屋舍，如地藏庵[1]。
    • 巖：指位於山窟或依山崖處據險而築的寺廟[1]。
    • 洞：與「巖」相似。
    • 宮：原指帝王居所，傳統上只有主神神格為帝後或王爺級的寺廟才會用宮字，如天后宮等[1]。
    • 府：原指百官所居之處[1]。
    • 亭：原是指讓人停留休息的建築[1]。

    神明標記方式

    依照神明類型，另外新增幾個標籤用以標注 - 

    • 主祀神 (統一神、一般神、地方神、雜神，或行政神、司法神、職業神)、
    • 屬祀神 (配偶神、配祀神、挾祀神、隸祀神、分身神)、
    • 客祀神 (同祀、寄祀)。 

    如

    dedication=土地公

    dedication:principal=中壇元帥 # 主祀神

    dedication:patron=五營元帥 # 屬祀神 

    dedication:guest=虎爺公 # 客祀神 

    如屬神有數個，可以用分號分隔。

    historic=wayside_shrine

    以 wikidata 連結資料

    神明名稱可用原生語言。但是由於傳統宗教的神明往往有許多別名，且原生語言會造成鄰近區域但不同語系的使用者使用上的困難，例如中國使用簡體字、或是越南信仰[5][6]一樣的神明。

    例如越南河內真武觀 (Đền Quán Thánh) 祭拜 Trấn Vũ，即為玄帝、黑帝、真武大帝、玄天上帝、玄武大帝[7][8][9][10]等等名稱。

    因此建議使用 wikidata tag 來標注對應神明身份，例如

    dedication:wikidata=Q2670522 # 土地公

    dedication:principal:wikidata=Q547105 # 主祀神 中壇元帥

    dedication:patron:wikidata=Q10881976 # 屬祀神  五營元帥  

    dedication:guest:wikidata=Q15903662  # 客祀神 虎爺公
