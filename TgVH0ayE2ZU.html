  2. Download pbf, xml, shp fiels from planet , or download only Taiwan area data from geofabrik.
  3. Generate and print the map with Maperitive. 
  4. Use overpass turbo to access interestedPOI and export as GeoJSON.
  5. How to use and access to customized vector map by own server or commercial service like mapbox.
