飲水地圖線上回報功能

用 OSM Note 來做

  • 填資料範例 POST http://www.openstreetmap.org/api/0.6/notes?lat=24.00&lon=121.00&text=ThisIsANote
    • 這 API 有點危險 XD
  • 讀資料範例 GET https://www.openstreetmap.org/api/0.6/notes.json?bbox=121.4564323425293,24.974854621062875,121.5084457397461,25.073782645952882
  • 文件 http://wiki.openstreetmap.org/wiki/API_v0.6#Map_Notes_API
  • 可以新增、comment、關閉、reopen、search

作法

  • 網站上做一個表單，填名稱、水溫、室內／室外、樓層、說明（再參考之前整理的 tag 表）、「這是準確位置還是大略的位置？（checkbox，預設是「大略的」）」
  • 地圖上要加一個準星，避免沒對準
  • 加一個 hashtag #飲水地圖
  • 幫使用者 POST 進去
  • 驗證碼
  • 做一個列出所有 #飲水地圖 的 Note 列表，然後訓練一批人專門解這種 Note
    • 也可以 POST 完以後把 id 存到 Google Spreadsheet/Ethercalc，可以請大家有空幫忙解

問題

  • OSM 底圖的資料不足
    • 用通用版電子地圖？
  • 需要說一下寫 Note 注意事項？
