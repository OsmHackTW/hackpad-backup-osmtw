動物園資訊

交通方式

  • 大眾運輸：
    • 56號壽山動物園專線公車
      • 由火車站出發→捷運鹽埕埔站→動物園→捷運鹽埕埔站→火車站
      • 捷運鹽埕埔站搭乘地點: 4號出口至五福四路華南銀行前搭乘
      • 發車時間：
        • 每日上午08:30至17:30，平日每小時一班，假日每15-20分鐘一班。（周一休園停駛）
        • 夜間延長開放期間56號公車配合延長發車時間，動物園大門末班車發車時間為晚間9點，歡迎民眾多加利用。
    • 219號
      • 加昌站←→加昌路←→後昌路←→左楠路←→左營大路←→鼓山行政中心站 ←→大公路←→臨埕圓環站 鼓山行政中心站下車，步行萬壽路上山，經元亨寺、萬壽山橋及動物園停車場即可抵達動物園

圖資收集

工具

  • 電腦
  • GPS device (動物園通常樹都很多，空照圖英雄無用武之地)
  • Mapbox 的圖資應該能補木欄動物園一帶
  • 壽山動物園的 bing 其實很清楚，但是步道上都是樹
  • 園方導覽地圖
  • 紙筆
  • Fieldpapers
    • http://fieldpapers.org/atlases/43l5av3b (全區共兩格)
    • http://fieldpapers.org/atlases/5i8f8vhy (東側共三格)
    • http://fieldpapers.org/atlases/25d8aq96 (西側共兩格)
  • 相機 (紀錄台階，出入口等等)
  • 手機裝 Mapillary 自製街景
  • 玩樂的心

重點 POI

  • 洗手間 amenity = toilet 
  • 飲水機 amenity = drinking_water (細節: http://beta.hackfoldr.org/drinking-water)
  • 自動販賣機 amenity = vending_machine 
  • 禮品部 shop = gift 
  • 各式各樣的資訊站 http://wiki.openstreetmap.org/wiki/Information
  • 哺乳 http://wiki.openstreetmap.org/wiki/Proposed_features/babycare 
  • AED emergency = defibrillator 
  • 滅火器 emergency=fire_extinguisher 
  • 消防栓  emergency=fire_hydrant 
  • 消防水龍  emergency=fire_hose 

Tagging schema

觀察 Berlin zoo 的方式，每個分區的標法如下 (範例 )：

  • tourism = attraction
  • attraction = animal
  • name = 分區名
  • building = ... (optional)
