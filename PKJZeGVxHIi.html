倒資料

包含自動或手動依據某些現存資料大量輸入資料到 OpenStreetMap 而未完全依循 Import guidline。

罪咎較輕者，請搭配第二次 Webinar 的簡報 服用。罪孽深重者，請詳讀 Import Guideline 。

也可以聽聽看 Lucien  在 COSCUP 2014 分享的議題尋求解套。

常見錯誤有：

  • 未在正式 import 之前將方法和來源向社群說明
  • 資料來源授權有爭議
  • 資料來源正確性未經驗證
