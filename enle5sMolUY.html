如何疊上合適的空照圖

這裡介紹如何在 iD 和 JOSM 疊上合適的空照圖或其他 wms, tms

使用 iD

  • 申請帳號之後，在 http://www.openstreetmap.org 開啟 iD editor 
  • 畫面右邊的「背景圖像設定」

  • 點選「客製化」 
  • 輸入空照圖的 URL

使用 JOSM

這裡介紹如何透過 JOSM 新增 WMS 圖層或是疊合其他點陣圖

支援格式

  • JOSM 本身就可以使用 WMS 或 TMS 的圖層
  • 透過 plug-in ，JOSM 又可以支援其他點陣圖或是 *.shp file 等等。

WMS or TMS

*.shp file

空照圖URL範例

  • 福衛災前 (FormoSAT-2 before explosion)
    • for iD
      • http://gis.sinica.edu.tw/googlemap/Formosat_Taiwan_2013/{17-z}/{x}/IMG_{x}_{y}_{17-z}.png
    • for JOSM
      • tms[23]:http://gis.sinica.edu.tw/googlemap/Formosat_Taiwan_2013/{17-zoom}/{x}/IMG_{x}_{y}_{17-zoom}.png
  • 農航所災前 (Arial photo before explosion)
    • for iD
      • http://maps.nlsc.gov.tw/S_Maps/wmts?SERVICE=WMTS&REQUEST=GetTile&VERSION=1.0.0&FORMAT=image/png&STYLE=_null&LAYER=PHOTO2&TILEMATRIXSET=EPSG:3857&TileMatrix={z}&TileCol={x}&TileRow={y}
    • for JOSM
      • tms:http://maps.nlsc.gov.tw/S_Maps/wmts?SERVICE=WMTS&REQUEST=GetTile&VERSION=1.0.0&FORMAT=image/png&STYLE=_null&LAYER=PHOTO2&TILEMATRIXSET=EPSG:3857&TileMatrix={zoom}&TileCol={x}&TileRow={y}
